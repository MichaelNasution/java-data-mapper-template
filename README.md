# java-data-mapper-template
Template arsitektur bersih berbasis Java yang mengintegrasikan JDBC, Java Collection Framework (JCF), inheritance, serta ORM menggunakan pola Data Mapper. Proyek ini dirancang sebagai fondasi reusable untuk membangun sistem backend yang terstruktur, scalable, dan mudah dipelihara.


Template proyek Java dengan pendekatan clean architecture yang mengimplementasikan JDBC, Java Collection Framework (JCF), inheritance, dan ORM berbasis Data Mapper. Dirancang sebagai fondasi umum untuk pengembangan sistem backend yang terstruktur, modular, dan mudah dikembangkan.
